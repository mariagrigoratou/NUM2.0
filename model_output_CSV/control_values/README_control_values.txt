This folder includes the data used for the Figures 1 and 2. 
The csv files include the daily and seasonal mean of Sea Surface Temperature (SST) and Mix Layer Depth (MLD) as used Grigoratou et al., 2024. The biomass concentrations for copepods and protists (seasonal, mean) are the model output with the environmental forcing used Grigoratou et al., 2024.
